생성된 Suno 프롬프트

🎚️ Logic Pro로 보내기

Suno에서 받은 오디오 파일을 바탕화면 폴더에 모아두면, 그 폴더 안의 파일들을 이름 순서대로 Logic Pro에 자동으로 가져오는 명령을 만들어드려요. 브라우저 보안 정책상 웹페이지가 직접 터미널 명령을 실행할 수는 없어서, 복사해서 터미널에 붙여넣을 명령어를 안내합니다.

~/Desktop/

  1. 손쉬운 사용 권한 (최초 1회) — 시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용에서 터미널 앱에 권한을 허용하세요. AppleScript가 Logic Pro 메뉴를 자동 클릭하려면 필요합니다.
  2. 폴더 준비 — 가져올 오디오 파일을 위에서 지정한 폴더에 넣고, 재생 순서대로 "01_intro.wav, 02_verse.wav…"처럼 번호를 붙여 이름을 정렬해두세요.
  3. Logic Pro 프로젝트 열기 — 파일을 가져올 프로젝트를 Logic Pro에서 미리 열어두세요. (스크립트는 새 프로젝트를 만들지 않아요)
  4. 터미널에서 실행 — 터미널 앱을 열고 아래 명령어를 붙여넣어 실행하세요.

생성 히스토리

아직 저장된 히스토리가 없어요. 프롬프트를 생성하면 여기에 자동으로 기록됩니다.
